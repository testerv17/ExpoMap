# Target AR (MindAR) — Cómo generar el archivo .mind

Target (imagen):
- `assets/targets/expo_target.jpg`

MindAR requiere un archivo compilado `.mind`.

## Cómo generar
1) Busca: **MindAR image target compiler**
2) Sube `expo_target.jpg`
3) Compila / Generate
4) Descarga el `.mind`
5) Renómbralo a `expo_target.mind`
6) Colócalo en: `assets/targets/expo_target.mind`
